# Saku

Aplikasi catatan uang saku sederhana berbasis HTML/CSS/JavaScript.

## Repository GitHub

- Repository: https://github.com/m.aziz/websakubulanananakkost
- GitHub Pages: https://m.aziz.github.io/websakubulanananakkost/

## Cara deploy ke GitHub Pages

1. Buat repository GitHub baru.
2. Upload semua file dari folder ini ke repository.
3. Buka Settings > Pages.
4. Pilih branch `main` atau `master` dan folder `/root`.
5. Tunggu beberapa menit sampai GitHub memberikan URL publik.
